Add the following computed getters to the Alpine store, after the today/yesterday getters added in Plan 01:

1. **_peakDay(metricFn) helper** — generic reducer that finds the day with the highest value for a given metric extractor function. Scans `this.timeseries.days` (ALL days, not filteredDays). Returns the full day object or null if no data.
   ```
   _peakDay(fn) {
     if (!this.timeseries || !this.timeseries.days || !this.timeseries.days.length) return null;
     return this.timeseries.days.reduce((best, d) => fn(d) > fn(best) ? d : best);
   }
   ```

2. **_peakProject(day, metricKey) helper** — extracts the top project name from a day's byProject data for the given metric. Guard for missing byProject: return null if not present. For tokens, sum the token sub-fields. For cost, use costCents.
   ```
   _peakProject(day, metricKey) {
     if (!day || !day.byProject) return null;
     let best = null, bestVal = -1;
     for (const [name, stats] of Object.entries(day.byProject)) {
       let val;
       if (metricKey === 'tokens') val = stats.tokens.input + stats.tokens.output + stats.tokens.cacheCreate + stats.tokens.cacheRead;
       else if (metricKey === 'cost') val = stats.costCents;
       else val = stats[metricKey];
       if (val > bestVal) { bestVal = val; best = name; }
     }
     return best;
   }
   ```

3. **_fmtShortDate(dateStr) helper** — formats "2026-03-15" as "Mar 15":
   ```
   _fmtShortDate(dateStr) {
     if (!dateStr) return '';
     const d = new Date(dateStr + 'T12:00:00'); // noon to avoid timezone shift
     return d.toLocaleDateString('en-US', { month: 'short', day: 'numeric' });
   }
   ```

4. **Four peak getters**, each returning an object `{ value, date, project }` or null:

   - `get peakMessages()` — uses `d => d.messages`, metricKey 'messages', formats value with `_fmtNum`
   - `get peakSessions()` — uses `d => d.sessions`, metricKey 'sessions', formats with `_fmtNum`
   - `get peakTokens()` — uses the `_totalTokens(d)` helper as metric extractor, metricKey 'tokens', formats with `_fmtNum`
   - `get peakCost()` — uses `d => d.costCents`, metricKey 'cost', formats value as `$X.XX` (exact, no tilde prefix — this is a historical record)

   Each returns: `{ value: formattedString, date: _fmtShortDate(day.date), project: _peakProject(day, key) }`
   If no data, return null.
