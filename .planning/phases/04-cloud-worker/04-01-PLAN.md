  <action>
    **src/kv.ts:** KV helper module with typed functions wrapping raw KV calls:
    - `getCardCache(kv: KVNamespace, username: string, theme: string, layout: string, style: string): Promise<string | null>` — reads `card:{username}:{theme}:{layout}:{style}` key
    - `putCardCache(kv: KVNamespace, username: string, theme: string, layout: string, style: string, svg: string): Promise<void>` — writes the same key pattern, NO expirationTtl (cache until sync)
    - `getUserData(kv: KVNamespace, username: string): Promise<SafeStats | null>` — reads `user:{username}:data`, parses JSON
    - `putUserData(kv: KVNamespace, username: string, data: SafeStats): Promise<void>` — writes user data as JSON
    - `invalidateCardVariants(kv: KVNamespace, username: string): Promise<number>` — lists `card:{username}:` prefix, deletes all matching keys, returns count. Single page list (max 18 variants = 3 styles x 2 themes x 3 layouts, well under 1000 limit)
    - `deleteAllUserData(kv: KVNamespace, username: string): Promise<void>` — deletes user data key + all card variants
    - `getTokenUsername(kv: KVNamespace, token: string): Promise<string | null>` — reads `token:{token}:username`
    - `putToken(kv: KVNamespace, token: string, username: string): Promise<void>` — writes token with 1 year expirationTtl
    - `deleteToken(kv: KVNamespace, token: string): Promise<void>` — removes token

    **src/routes/card.ts:** Export a Hono sub-app for the card route:
    ```
    const cardRoutes = new Hono<{ Bindings: Env }>()
    cardRoutes.get('/:username', async (c) => { ... })
    ```

    Handler logic:
    1. Extract `username` from path param
    2. Extract query params: `theme` (default 'dark'), `layout` (default 'classic'), `style` (default 'github')
    3. Check KV cache via `getCardCache()` — if hit, return SVG with correct headers
    4. If cache miss, fetch user data via `getUserData()`
    5. If no user data, return `renderPlaceholderCard(username)` with correct headers
    6. If user data exists, call `renderCard(userData, { theme, layout, style })` to generate SVG
    7. Store in KV cache via `putCardCache()`
    8. Return SVG

    CRITICAL: Every SVG response (cached, rendered, placeholder) MUST include these headers:
    ```
    Content-Type: image/svg+xml
    Cache-Control: no-cache, no-store, must-revalidate
    Pragma: no-cache
    Expires: 0
    ```
    Extract header creation into a helper: `svgResponse(c, svg: string, status = 200)`.

    **src/index.ts:** Hono app entry point:
    ```
    import { Hono } from 'hono'
    import type { AppType } from './types.js'
    import { cardRoutes } from './routes/card.js'

    const app = new Hono<AppType>()

    // Health check
    app.get('/', (c) => c.json({ name: 'shiplog-worker', status: 'ok' }))

    // Card serving
    app.route('/card', cardRoutes)

    export default app
    ```

    Do NOT add auth or sync routes yet — those are Plan 02.
